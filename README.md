# Docker

## Docker Commands

| Command                                                     |               Action                                          |
|-------------------------------------------------------------|---------------------------------------------------------------|
|`docker ps`                                                  | docker running containers                                     |
|`docker ps -a`                                               | docker exited containers                                      |
|`docker run -it microsoft/windowsservercore cmd`             | run windowsservercore image, run command inside container     |
|`docker run -it microsoft/windowsservercore cmd`             | run windowsservercore image, run command inside container     |
|`docker run -it microsoft/windowsservercore cmd`             | run windowsservercore image, run command inside container     |
